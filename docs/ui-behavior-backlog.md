- **第 1 行**：用户已决定**暂不在 Settings 增 `statsPrefs` 字段**，故其数据真值「暂未进入 Settings」；待统计页设计阶段再决定是否持久化展示偏好及存储方式。

- **未纳入的纯性能项**：`appDate` 落库性能优化评估（原 §14【37】）已迁入 §11，**不**进本文件（非 UI / 行为问题）。
- **多端云端备份（§14【18】）**为 Phase 5+ 范围，既不进 §11 也不进本文件。
- 本文件与 §11 为**派生视图关系**，非替代：§11 仍是正文内的后置实现索引；本文件把其中 UI / 行为 / 展示类条目 + §14 UI 面集中成一处便于产品侧跟踪。两者如有出入，以 `data-layer-spec-v4.md` 正文为准。
